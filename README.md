## Formato de Nomeclatura BEN(Block Element Modifier)

- É uma forma de escreve o nomes das `class`, `id` e afins.

- Exemplos:

    ```html

        <section class="main__section">
            <h1 class="main__text"> Olá </h1>
        </section>

    ```

- Resultado :

    - Aqui utilizamos o modeo `BEN` para escreve a `section` e atribuímos o `main__section` a ela, ela pertece ao bloco `main`, note que em nosso `h1` foi atribuído o mesmo bloco `main__text` e logo após o seu tipo que é `text`, isso é o formado `BEN` para estilização no `css` e padronização no `html`.
